# 用語辞書（BBS vNext）

`.idea/` 内資料の語彙を、実装/画面/ドキュメントで使うために整理した辞書。

## 基本用語

- `ぁ界`: あやしいわーるど全体の文化圏を指す語
- `ぁ民`: 利用者（住人）
- `固定`: ハンドルを継続利用する投稿者
- `空白 / 名無し`: 名前欄を空欄で投稿する利用形態
- `ROM`: 書き込まず閲覧する行為

## 実装寄り用語（本プロジェクト）

- `owner_key`: セッション内で投稿者本人性を判定するキー
- `owner_key_hash`: 投稿に保存するハッシュ化済み所有者情報
- `未読返信`: 自分の投稿に対する、未既読の返信
- `NG`: 非表示判定に使う語句設定

## UI表記ルール

- 表記は短く保つ（例: `NG`, `未読返信`, `絞り込み`）
- 技術語は画面に出さない（例: `owner_key_hash` は非表示）
- 既存表記を優先して統一する

## 命名ガイド（コード）

- セッションキー: `snake_case`（例: `read_reply_ids`）
- View変数: 意味が推測できる英語名（例: `unreadReplyCount`）
- ルートクエリ: 短く機能が明確な名前（例: `mark_read_reply_id`）
