# 文言ガイド（BBS vNext）

`.idea/` 内資料をもとに、UI/通知/ヘルプ文のトーンを統一するためのガイド。

## 目的

- 掲示板の世界観を維持する
- 文言のブレを減らす
- 新機能追加時の命名ルールを明確にする

## トーン

- 基本は「簡潔・口語・やわらかめ」
- 説明は短く、命令口調を避ける
- 過度に煽らない
- 既存の雰囲気（深緑背景/匿名文化）と整合する語彙を使う

## 推奨表現

- 未読返信: `未読返信`
- 返信通知: `あなたの投稿に返信があります`
- 絞り込み: `絞り込み`
- NG設定: `NG`
- 解除操作: `〜だけ解除`

## 非推奨表現

- 管理画面的すぎる表現（例: `フィルタパラメータ`）
- 現代SNS寄りすぎる表現（例: `タイムライン`）
- 強い断定・攻撃的な文言

## 例（Before / After）

- Before: `フィルタをクリア`
- After: `絞り込みだけ解除`

- Before: `新着メッセージがあります`
- After: `未読返信があります`

## 運用ルール

- 新規文言追加時は、このファイルに追記してから実装する
- 既存文言を変更する場合は、影響範囲（画面/通知/README）を同時確認する
