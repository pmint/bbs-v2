# 機能優先度表（vNext）

`.idea/` の文脈と現状の実装を踏まえた、BBS vNext の改善優先度。

## 直近反映済み

1. 会話の見える化
- 未読返信バー
- 返信ジャンプ
- 既読管理（セッション）

2. 投稿文脈の追跡性
- 返信元/返信先の導線
- スレッド表示
- 自分の投稿と自分への返信の右寄せ表示

3. ハッシュタグと過去ログの基本改善
- 題名内 #タグ のクリック絞り込み
- タグ一覧集計
- 過去ログ検索結果からの返信/スレッド導線

4. 広報室の予定表示
- 利用者向けの「今後の予定」欄
- 実装済み項目と予定項目の切り分け
- 更新履歴との並び順整理

5. 通知体験の整理
- 未読返信バーを中心にした返信通知
- 新しい未読返信だけを一時通知で件数表示
- 同じ返信を過剰に通知しない表示ルール

## 優先度A（次スプリント対象）

1. 投稿支援
- 下書き・プレビュー補助
- 入力途中の内容を戻しやすくする導線

## 優先度B（A完了後）

1. 検索体験の強化
- 複合条件検索
- 検索条件の見直しやすさ改善

## 優先度C（将来検討）

1. 保存検索・履歴
- よく使う検索条件の保存
- 最近使った検索条件の再利用

2. 入力補完
- よく使うタグや表現の入力補助
- 世界観を壊さない範囲での補助表示

## 判断基準

- 世界観を壊さないか
- 実装コストに対して体験改善が大きいか
- セッション設計との整合が取れるか
- デプロイ運用に影響が少ないか
