# 機能優先度表（vNext）

`.idea/` の文脈を踏まえた、BBS vNext の改善優先度。

## 優先度A（次スプリント対象）

1. 会話の見える化
- 未読返信バッジ
- 返信ジャンプ
- 既読管理（セッション）

2. 投稿文脈の追跡性
- 返信元/返信先の分かりやすい導線
- スレッド内での参照しやすさ改善

## 優先度B（A完了後）

1. ハッシュタグ体験の強化
- タグの見つけやすさ改善
- タグ絞り込み導線の最適化

2. 通知体験の改善
- 通知表示ルールの整理
- 過剰通知の抑制

## 優先度C（将来検討）

1. 高度検索
- 複合条件検索
- 履歴・保存検索

2. 投稿支援
- 下書き・プレビュー補助
- 入力補完

## 判断基準

- 世界観を壊さないか
- 実装コストに対して体験改善が大きいか
- セッション設計との整合が取れるか
- デプロイ運用に影響が少ないか
